# TechInspector
A sleek, real-time system telemetry and hardware diagnostics dashboard built with Python (Flask).
Developed by the **TechNodez** team.

## How to Run

**Method 1: Using the Executable (Standalone)**
1. Run the `TechInspector.exe` file (starts a local server).
2. Open your web browser.
3. Navigate to `http://127.0.0.1:5000` to view the live dashboard.

**Method 2: Running from Source (Python)**
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python main.py`
3. Open `http://127.0.0.1:5000` in your browser.
